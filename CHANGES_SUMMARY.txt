----------------------------------------------------------------
YESTERDAY'S CHANGES
----------------------------------------------------------------

1. INVOICE PRINT REDESIGN
   - Compact A5 landscape layout (was A4)
   - 3-column header: From | Bill To | Job Details
   - 2-column bottom: Payment Info + Terms | Summary Table
   - Added "Download PDF" button using html2pdf.js
   - Shop logo displayed in invoice header (if uploaded)

2. COLLECT PAYMENT CAP
   - Payment amount in invoice section cannot exceed outstanding balance
   - Shows real-time balance remaining as user types

3. WHATSAPP URL FIX
   - All WA buttons updated from whatsapp://send to https://wa.me/91...
   - Fixed in: jobs.html, invoices.html, customers.html

4. REFUND ON CANCELLATION
   - Cancel modal redesigned as "Return & Refund" form
   - Fields: Job ID, Customer, Advance Paid (read-only),
             Refund Amount (read-only), Refund Method, Reason
   - New DB columns: refund_amount, refund_method, refund_date
   - New route: POST /jobs/<id>/record_refund

5. WHATSAPP MESSAGE ON CANCEL
   - Auto-triggers WA message after confirming cancellation
   - Message includes: customer name, device, job ID,
     advance paid, refund amount/method, UPI note if applicable
   - window.open() called BEFORE await to avoid browser popup block

6. SECRET_KEY PERSISTENCE
   - Changed from os.urandom(24) to os.environ.get('SECRET_KEY')
   - Prevents session invalidation on every server restart
   - Set SECRET_KEY in Railway environment variables

7. ADMIN CREDENTIALS VIA ENV VARIABLES
   - ADMIN_EMAIL, ADMIN_PHONE, ADMIN_PASSWORD can be set via env
   - Falls back to: admin@mobilefix.com / 0000000000 / admin123

8. USER MANUAL
   - Created templates/user_manual.html
   - 14 sections covering all features
   - Route: /manual

9. DUE AMOUNT IN INVOICES
   - Due date field appears when partial payment entered
   - Stored in invoices.due_date column
   - Shown in amber on invoice list under Balance Due

TODAY'S CHANGES
----------------------------------------------------------------

1. BALANCE DUE FIX IN CUSTOMER POPUP

   - "Set Reminder" now opens /jobs/<id>/set_reminder in new tab
   - Dedicated full-page form with job details + date picker
   - Created: templates/set_reminder.html
   - Updated: app.py (GET + POST route)

4. COMPANY LOGO UPLOAD IN SETTINGS
   - New logo upload field in Shop Settings
   - Preview shown instantly before saving
   - Stored as base64 data URL in users.logo DB column
   - Displayed in invoice print header
   - Files: templates/settings.html, app.py

   - Added migration: UPDATE users SET trial_start=datetime('now')
     WHERE trial_start IS NULL

   - File: app.py
